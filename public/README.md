# Public Routes Overview

Active pages:
- `index.php` (homepage)
- `login.php`, `register.php`, `logout.php`, `user.php` (auth/user flow)
- `how.php`, `helps.php` (info pages)

Compatibility route:
- `profile.php` -> redirects to `user.php`

Admin routes:
- `admin/admin.php` (admin dashboard + AI test panel)
- `admin/admin_raw.php`, `admin/admin_ai.php` (raw/AI detail views)
- `admin/openai_test.php` (AJAX endpoint for admin test panel)
